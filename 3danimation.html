Neo-Tokyo Bike

During the creation of an unfinished project, the idea was to create multiple vehicles that would be part of a “kart” type experience, but the model also had to feel as part of the world's lore.

The aim of this project was for it to be set in a dystopian post-apocalyptic world, this opened opportunities of expanding the racetracks, vehicles and locations and with each vehicle we had a character and lore. This bike was supposed to be a futuristic motorbike that was built in New Tokyo, a cyber-punk city. The design of this model was inspired from Akira, including the spiked tires, although this initially did not correlate with the theme, but the character’s backstory justifies the design.


D-Scales Truck

This was also part of the unfinished “kart” project. The model of this truck was inspired by the film Mad Max: Fury Road with the truck being rusted and modified, with in-built weapons that would attack the other contestants in the race and the reptilian elements on the vehicle was correlated to the character’s lore who was an alligator humanoid figure and the track race was set to be in Colorado.


Flame Boss

The flame boss, which is an enraged spirit that threatens the spirit world, starts of as an entity but as you progress throughout the story; it shall reveal itself. The design is kept simple and clean with UV mapping added and, in the future, textured.

Cardboard Man

This character was designed for my own personal project. I wanted to create a basic character that would have a set of animations of it fighting, using sword and shield. This model has a fully fleshed out UV mapping.
